
# 北京介绍
北京是中华人民共和国首都，位于华北平原北部。国家中心城市和政治文化中心。面积16410平方千米，人口约2180万。北京是世界著名古都，拥有3000多年建城史和860多年建都史。属温带季风气候，四季分明。作为历史文化名城，北京拥有故宫、天坛等众多世界文化遗产。

# 北京美食
北京美食集北方饮食之大成。北京烤鸭以色泽红艳、肉质细嫩著称。炸酱面面条劲道，炸酱香浓。其他特色包括涮羊肉、豆汁焦圈、卤煮火烧和驴打滚，展现了京味饮食的丰富多样。

# 北京景点
北京景点以历史遗迹为核心。故宫是世界上现存规模最大、保存最完整的木质结构古建筑群。天坛是明清两代皇帝祭天场所，建筑精美。其他景点包括八达岭长城、颐和园、天安门广场和南锣鼓巷迹为核心。故宫是世界上现存规模最大、保存最完整的木质结构古建筑群。天坛是明清两代皇帝祭天场所，建筑精美。其他景点包括八达岭长城、颐和园、天安门广场和南锣鼓巷

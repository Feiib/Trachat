
# 沈阳介绍
沈阳是辽宁省省会，别称盛京、奉天，位于东北平原南部。国家历史文化名城，东北地区重要中心城市。面积12948平方千米，人口约915万。沈阳是清朝发祥地，素有"一朝发祥地，两代帝王都"之称。工业基础雄厚，有"东方鲁尔"美誉。属温带季风气候，地势平坦，辽河贯穿全境。

# 沈阳美食
沈阳美食融合满族和东北饮食特色。老边饺子选料讲究，煸馅工艺独特。李连贵熏肉大饼熏肉香醇，大饼酥脆。锅包肉外焦里嫩，酸甜可口，是经典东北菜。其他特色包括白肉血肠、小鸡炖蘑菇、猪肉炖粉条、马家烧麦和西塔大冷面，体现了粗犷实在的关东饮食风格。

# 沈阳景点
沈阳景点以清朝历史遗迹为主。沈阳故宫是清朝初期皇宫，中国现存两大宫殿建筑群之一。张氏帅府是张作霖和张学良官邸，兼具中西建筑风格。北陵公园包含清昭陵及现代园林景区。九一八历史博物馆记录日本侵华历史。其他景点包括东陵公园、中街商业区、西塔韩国城和辽宁省博物馆。

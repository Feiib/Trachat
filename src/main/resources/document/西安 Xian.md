# 西安介绍
西安是陕西省省会，古称长安、镐京，位于关中平原中部。世界历史文化名城，中国四大古都之一。面积10096平方千米，人口约1300万。西安是丝绸之路起点，历史上包括周、秦、汉、唐在内的13个王朝在此建都。现存兵马俑、大雁塔、城墙等众多历史遗迹，被誉为"天然历史博物馆"。属温带季风气候，地形南依秦岭，北临渭河。

# 西安美食
西安美食体现西北饮食特色，以面食和清真风味为主。羊肉泡馍将掰碎的饦饦馍与羊肉汤同煮，配以糖蒜和辣酱。肉夹馍使用白吉馍夹入腊汁肉，外酥里嫩。凉皮分为米皮和面皮，配以辣油和醋水。其他特色包括biangbiang面、葫芦头、灌汤包子、甑糕和柿子糊塌，展现了关中地区悠久的饮食文化传统。

# 西安景点
西安景点以历史遗迹为核心。兵马俑博物馆展示秦始皇陵陪葬坑中的陶俑军阵，规模宏大。西安明城墙是中国现存规模最大、保存最完整的古代城垣。大雁塔为唐代高僧玄奘保存佛经而建，是佛教建筑杰作。钟鼓楼位于城市中心，是明清时期的标志性建筑。其他景点包括华清池、碑林博物馆、大明宫遗址公园和回民街。


# 成都介绍
成都是四川省省会，别称蓉城，位于四川盆地西部。西南地区科技、商贸、金融中心。面积14335平方千米，人口约2120万。成都历史悠久，有"天府之国"美誉，是古蜀文明发祥地。熊猫主要栖息地，美食文化闻名遐迩。属亚热带湿润气候，物产丰富，都江堰水利工程保障农业发达。

# 成都美食
成都美食以麻辣味型为主，是川菜代表。火锅使用牛油锅底，配以多样食材。夫妻肺片由牛杂凉拌而成，麻辣浓香。担担面面条细薄，臊子酥香。其他特色包括麻婆豆腐、龙抄手、钟水饺、赖汤元和串串香，体现了四川饮食文化的丰富多样性。

# 成都景点
成都景点融合自然与人文景观。都江堰是古代水利工程，至今仍在发挥作用。青城山为道教发源地，山林幽深。宽窄巷子保留清代街区格局，商铺林立。大熊猫繁育研究基地可近距离观察熊猫。其他景点包括武侯祠、杜甫草堂、锦里古街和金沙遗址博物馆。

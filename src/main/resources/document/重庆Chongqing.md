# 重庆介绍

重庆是中华人民共和国直辖市，位于中国西南部、长江上游地区。简称渝，别称山城、江城、雾都。面积8.24万平方千米，人口约3190万。重庆是巴渝文化发祥地，有3000余年建城史，宋光宗赵惇赐名“重庆”，取“双重喜庆”之意。抗战时期为国民政府陪都，1997年成为直辖市。地形以丘陵、山地为主，属亚热带季风气候，长江与嘉陵江在此交汇。

# 重庆美食

重庆美食以麻辣鲜香为特色，是川菜的重要分支。火锅为最具代表性美食，以牛油锅底、九宫格格式和毛肚、黄喉等鲜货食材闻名。重庆小面是日常主食，面条劲道、汤汁红亮。酸辣粉以红薯粉加工而成，酸辣开胃。其他特色包括毛血旺、辣子鸡、串串香、抄手和豆花饭，体现了地方饮食文化的丰富性。

# 重庆景点

重庆景点融合自然与人文景观。洪崖洞为悬崖吊脚楼建筑群，夜景灯光璀璨，被誉为现实版《千与千寻》。解放碑是市中心地标，纪念抗战胜利。磁器口古镇保留老码头街道格局，商铺林立。大足石刻是世界文化遗产，展示唐宋佛教雕刻艺术。武隆喀斯特旅游区以天生三桥、仙女山等喀斯特地貌著称。其他景点包括长江索道、白公馆、渣滓洞和南山一棵树观景台。

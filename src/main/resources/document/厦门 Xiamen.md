# 厦门介绍
厦门是福建省副省级市，别称鹭岛，位于中国东南沿海、台湾海峡西岸。经济特区和港口风景旅游城市。面积1700平方千米，人口约530万。厦门由厦门岛、鼓浪屿及众多小岛组成。海洋性气候特征明显，四季温暖湿润。作为闽南地区重要城市，厦门是著名的侨乡和闽南文化代表区域。

# 厦门美食
厦门美食融合闽南与南洋风味。沙茶面汤底微辣中带香甜，是头号美食。海蛎煎使用当地盛产的海蛎，口感软糯嫩滑。其他特色包括姜母鸭、土笋冻、花生汤和烧肉粽，体现了海滨城市的饮食特色。

# 厦门景点
厦门景点以海岛风光为主。鼓浪屿有万国建筑博览之称，是世界文化遗产。环岛路沿海岸线延伸，风景优美。其他景点包括南普陀寺、厦门大学、曾厝垵和胡里山炮台。

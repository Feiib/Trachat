
# 深圳介绍
深圳是广东省副省级市，位于珠江三角洲东岸。经济特区和全国性经济中心城市，粤港澳大湾区核心引擎。面积1997平方千米，人口约1760万。深圳原为宝安县，1980年设立经济特区，创造了举世瞩目的"深圳速度"。高新技术产业发达，被誉为"中国硅谷"。属亚热带海洋性气候，地势东南高西北低，沿海多为平原台地。

# 深圳美食
深圳美食融合各地风味，体现移民城市特色。光明乳鸽皮脆肉嫩，汁水丰富。沙井鲜蚝肥美爽滑，可炭烤或清蒸。椰子鸡以海南椰青和文昌鸡烹制，汤清味甜。其他特色包括公明烧鹅、客家酿豆腐、盆菜、肠粉和潮汕牛肉火锅，展现了多元文化交融的饮食特点。

# 深圳景点
深圳景点现代化与自然风光并重。世界之窗微缩展示全球著名景观，是主题公园代表作。欢乐谷集游乐设施和表演于一体，适合家庭游玩。东部华侨城包含生态景区和度假设施。深圳湾公园沿海岸线延伸，是休闲运动好去处。其他景点包括大梅沙海滨公园、莲花山公园、华强北商业区和南头古城。

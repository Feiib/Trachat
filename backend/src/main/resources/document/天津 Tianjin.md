
# 天津介绍
天津是中华人民共和国直辖市，位于华北平原东北部，海河流域下游。国家中心城市和环渤海地区经济中心。面积11966平方千米，人口约1380万。天津自古因漕运而兴起，明朝永乐年间正式筑城，是中国古代唯一有确切建城时间记录的城市。属温带季风气候，四季分明。

# 天津美食
天津美食兼具北方特色与港口风味。煎饼馃子是代表性早点，以绿豆面浆摊制，配以鸡蛋和薄脆。狗不理包子以半发面皮和水馅打汤工艺闻名。其他特色包括耳朵眼炸糕、锅巴菜、罾蹦鲤鱼和麻花，展现了津门饮食的独特风格。

# 天津景点
天津景点以近代历史建筑为特色。五大道文化旅游区拥有规模庞大的万国建筑博览苑。意大利风情区保留原汁原味的欧式建筑群。其他景点包括天津古文化街、天津之眼摩天轮、盘山风景名胜区和独乐寺。


# 拉萨介绍
拉萨是西藏自治区首府，位于青藏高原中部。西藏政治、经济、文化和宗教中心。面积29518平方千米，人口约86万。拉萨是世界上海拔最高的城市之一，海拔3650米，素有"日光城"之称。藏传佛教圣地，有1300多年历史。属高原温带半干旱气候，日照时间长，昼夜温差大。藏族占总人口多数。

# 拉萨美食
拉萨美食体现藏族饮食特色。酥油茶由酥油和茶水搅拌而成，是日常饮品。糌粑以青稞面加工，可配酥油茶食用。牦牛肉做法多样，可风干或炖煮。其他特色包括藏面、甜茶、青稞酒、人参果酸奶和血肠，展现了高原地区独特的饮食传统。

# 拉萨景点
拉萨景点以宗教建筑为核心。布达拉宫是世界文化遗产，曾是达赖喇嘛冬宫。大昭寺供奉释迦牟尼12岁等身像，朝圣者络绎不绝。八廓街环绕大昭寺，是拉萨著名转经道。色拉寺以辩经活动闻名。其他景点包括罗布林卡、哲蚌寺、小昭寺和纳木错。

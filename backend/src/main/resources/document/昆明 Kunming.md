
# 昆明介绍
昆明是云南省省会，别称春城，位于云贵高原中部。中国面向东南亚的重要门户城市。面积21012平方千米，人口约860万。昆明属低纬高原山地季风气候，四季如春，年均气温15℃左右。少数民族聚居区，有彝、回、白、苗等世居民族。历史文化悠久，是滇文化发祥地之一，西南联大曾迁址于此。

# 昆明美食
昆明美食融合汉族与少数民族风味。过桥米线由汤料、主食和配料组成，吃法独特。汽锅鸡用特制陶锅蒸制，原汁原味。野生菌火锅以当地野生食用菌为主料，鲜美营养。其他特色包括官渡粑粑、烧饵块、火腿坨、豆花米线和宜良烤鸭，展现了云南多元的饮食文化。

# 昆明景点
昆明景点以自然风光和民族风情为主。石林是世界自然遗产，喀斯特地貌形态各异。滇池是云南最大高原湖泊，湖光山色壮丽。云南民族村展示25个少数民族文化风情。世博园集各国园林艺术精华。其他景点包括西山森林公园、金马碧鸡坊、大观楼和翠湖公园。

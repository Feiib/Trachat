# 广州介绍
广州是广东省省会，别称羊城、花城，位于珠江三角洲北部。国家中心城市和综合性门户城市，是粤港澳大湾区的核心引擎之一。面积7434平方千米，人口约1880万。广州是广府文化发祥地，有2200多年建城史，自古就是海上丝绸之路起点和重要贸易港口。属海洋性亚热带季风气候，温暖多雨，四季常青。

# 广州美食
广州美食以粤菜为主，讲究清淡鲜美。广式早茶品类繁多，包括虾饺、烧卖、凤爪等点心。竹升面用传统竹竿打压制成，口感爽滑劲道。啫啫煲食材在煲中啫啫作响，香气四溢。其他特色包括广式烧腊、白切鸡、煲仔饭、糖水和老火靓汤，体现了岭南饮食文化的精致细腻。

# 广州景点
广州景点融合现代与传统特色。白云山是城市绿肺，可登山远眺市区景色。广州塔为城市地标建筑，俗称小蛮腰。沙面岛保留近代欧陆风情建筑群。其他景点包括陈家祠、越秀公园、中山纪念堂和北京路步行街。

# 杭州介绍
杭州是浙江省省会，古称临安、钱塘，位于中国东南沿海、钱塘江下游。长江三角洲中心城市之一，著名的风景旅游城市。面积16850平方千米，人口约1250万。杭州是吴越国和南宋的都城，素有"人间天堂"美誉，以西湖文化景观和茶文化闻名。京杭大运河终点，互联网经济发达，拥有阿里巴巴等知名企业。属亚热带季风气候，四季分明，水网密布。

# 杭州美食
杭州美食属浙菜分支，讲究清淡鲜嫩。西湖醋鱼选用草鱼烹制，酸甜适口。东坡肉以猪肉加酒料焖制，酥而不碎。龙井虾仁配以龙井茶叶，清新爽口。其他特色包括叫化童鸡、干炸响铃、片儿川、定胜糕和葱包烩，体现了江南饮食文化的精致典雅。

# 杭州景点
杭州景点以西湖为核心。西湖风景区包含苏堤春晓、断桥残雪等十景，湖光山色相映。灵隐寺为江南著名古刹，始建于东晋时期。西溪国家湿地公园是城市湿地，生态资源丰富。千岛湖由人工水库形成，岛屿星罗棋布。其他景点包括雷峰塔、六和塔、宋城和河坊街。

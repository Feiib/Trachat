
# 苏州介绍
苏州是江苏省地级市，古称吴，别名姑苏，位于长江三角洲中部。国家历史文化名城，重要风景旅游城市。面积8657平方千米，人口约1290万。苏州有近2500年建城史，春秋时期为吴国都城，以园林艺术和水乡文化闻名。京杭大运河贯穿城市，形成水陆并行、河街相邻的双棋盘格局。属亚热带季风气候，地势低平，水网密布。

# 苏州美食
苏州美食讲究时令鲜嫩，口味偏甜，属苏帮菜系。松鼠鳜鱼造型美观，外脆里嫩，酸甜适口。阳澄湖大闸蟹膏黄肥美，是秋季时令佳肴。奥灶面以红油爆鱼面和白汤卤鸭面最为著名。其他特色包括碧螺虾仁、响油鳝糊、樱桃肉、生煎馒头和枣泥拉糕，体现了江南饮食的精致细腻。

# 苏州景点
苏州景点以古典园林和水乡古镇为特色。拙政园是中国四大名园之一，以水景营造见长。虎丘塔是苏州城市象征，有"吴中第一名胜"美誉。周庄古镇保存完整水乡建筑格局，小桥流水人家。留园建筑布局精巧，组合多变。其他景点包括狮子林、网师园、平江路历史街区和同里古镇。
